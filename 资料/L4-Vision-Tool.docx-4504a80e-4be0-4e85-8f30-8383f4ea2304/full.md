# 一、VisionPro Tools 应用

# 1. PMAlign 高级参数

$\textcircled{1}$ 弹性：提高弹性能增加粗糙得分，但是会降低精确度，一般取值范围放在 0-1，增加弹性可能会导致找到不准确的特征。最大不超过8  
$\textcircled{2}$ ② 颗粒度：粗糙特征(黄色)和精细特征(绿色) 值单位是像素，精细值一定低于粗糙值， 精细值越低，越能提取出更细微(占用很少像素)的特征   
③ 对比度阈值：定义模式实例的最小可接受对比度。PMAlign仅考虑所有特征边界上的像素值的平均差异超过对比度阈值的图案实例。对比度大可以适当提高这个值，对比度差可以适当降低  
$\textcircled{4}$ 自动边缘阈值：可启用 可不启用 启用时不使用自己设定的值

不勾选自动边缘阈值时，特征边缘的对比度小于设置值，匹配度较差显示红色，红色过多时匹配失败

勾选自动边缘阈值时，工具会设置一个系统自动匹配的阈值，文本框中阈值不启用

⑤XY 重叠： 0-1，表示两个图重叠度的百分比，实际重叠度小于设定的这个值才会显示出来结果，若为1则只要有可能重叠的结果都会显示出来  
$\textcircled{6}$ 计分时考虑杂斑：一般不勾选，尤其掩膜时考虑杂斑 会造成很难匹配到结果或是得分很低(因为图片很少会有和模板相似的杂斑项)

# 3. PatInspect 基础应用

# 操作方法：

1.构建相机像源，并采集一张较好的图像，然后再创建CogPMAlign工具，同时将其输入图像连接到像源  
2.配置 PMAlign 工具来定位图像中的物体，创建 PatInspect 工具，连接图像，将PMAlign 工具的 GetPose

输出端连接到 PatInspect 工具的 Pose 输入端，运行工具进行 Pose 信息的传递。

3.打开PatInspect工具编辑器，点击抓取训练图像和原点

4.切换到当前训练图像，框选需要检测的区域

5.点击训练新模式来创建一个训练模式

6.继续运行工具以获取新图像，返回编辑控件注意新的当前输入图像，单击统计训练当前模式，将新图像合并

到现有的训练模式中，此时运行参数选项卡中的训练模式图像窗口中的训练模式可能会发生轻微变化，此外

，编辑控件将更新用于创建这个训练模式的图像数量。

7.重复上述操作，对于添加的每个新图像，训练模式的变化也会越来越小，但最终应该得到一个训练模式，

该模式将生成包含真实缺陷的阈值差分图像。

8.当获得足够的图像来训练模式时，单击结束统计训练(End Statistical Training)，可结合其它工具进行进一步分析

PS：创建好训练模式后，既可以用来检测连续运行的图像，但是注意不同图像可能会存在一些光照的变化，

同时图像中的物体位置也可能轻微移动，因此需配置 PatInspect工具获得能够涵

盖这些细微变化的训练模式。

# 4.CNLSearch 基础应用

作用：是定位或测量事先训练的一个或多个特征的质量，即用来测量当前图像中某一天特征与先前训练的图模型的相似度。

应用：

定位：通过定位物体上的特征来确定已知物体的位置和方向

存在/不存在检测：验证图像中存在期望数量的特征。

测量：测量长度。直径、角度和其他关键尺寸，通过定位图像中的特征，然后计算它们之间的距离。

缺陷检测：搜索图像中的缺陷。只有在预先知道缺陷的出现时才有用。

线性模式搜索：如果指定搜索图像的亮度与模式图像的亮度线性不同，则 CNLSearch将查找搜索图像中像素值的模式与模式图像中像素值的模式最相似的部分。这种搜索称为强度相关搜索，因为搜索图像和模式图像之间的相似度是通过计算两幅图中灰度像素值模式之间的相关系数来确定的。CNLSearch 用于计算两幅图像之间的相关系数的方法不受图像之间亮度的线性变化的影响。

非线性模式搜索：如果指定搜索图像的亮度与模式图像的亮度非线性不同，则CNLSearch通过查找搜索图像中边缘和非边缘的图案与模式图像中边缘和非边缘的图案最相似的位置来搜索图像中的团。由于CNLSearch的非线性模式搜索的是边缘模式而不是像素值模式，因此CNLSearch对模式图像和搜索图像之间的线性和非线性亮度变化都免疫，只要亮度变化不影响搜索图像中的边缘模式。

工作原理：根据模式的训练图像在搜索图像中查找模式的位置。除了返回模式在搜索图像中的位置，它还通过返回分数来指示搜索图像中的模式与训练图像中的模式匹配的程度。分数表示训练图像与返回其位置的图像之间的匹配程度。得分范围从 0.0，表示模式和特征之间没有相似性，到1.0，表示完全匹配。除了返回在搜索图像中找到的最接近匹配的位置和分数，CNLSearch还可以返回图像中其他不太相似的模式实例的位置和分数。即使使用高性能CPU，对于大多数实际应用程序，彻底搜索也太慢，它使用了一种更复杂的技术来定位图像中的特征。首先，快速扫码图像以定位可能匹配的位置，然后只对这些候选位置进行相似性评估，返回最佳匹配的位置。

# 5. SearchMax 基础应用

是一个专门的搜索工具，结合了PMAlign和CNLSearch的特性，此工具允许训练要查找的图像功能的模式，然后在运行时图像中查找训练的功能的实例。

与CNLSearch一样，此工具使用规范化相关搜索来匹配运行时图像中训练模式的特征。

与PMAlign一样，允许在不同的旋转和缩放下查找特征实例。

与其他搜索工具不同，它还允许在RGB彩色图像中训练和定位特征。

对于大多数需要查找缩放和旋转特征实例的应用程序，PMAlign 比 SearchMax 更快、

更准确。

创建DOF模板(训练模式)：运行参数里的自由度选项会在训练参数里 同时 运行参数里需要勾选使用模板Dof

圧缩度：None无 Lossy：有损压缩 可选0-1压缩比 Lossless：无损压缩 针对模板的图片 采用有损压缩时得分也会变低使用压缩可显著提高速度，但可能会导致某些边缘模式实例被遗漏勾选有损压缩时，运行参数中 勾选使用计分的压缩 会使结果得分变低

对比度限制(运行参数)：eg： 0.5-1.5 指的是当对比度为 20 时， 限制范围为$2 0 ^ { * } 0 . 5 substack { -- 2 0 ^ { * } 1 . 5 }$

亮度限制(运行参数)：同上

混淆阈值：在CNLSearch和SearchMax中均存在混淆阈值，一般在图像中搜索模式的单个实例，只要找到一个分数高于混淆阈值的实例其就会停止搜索并返回匹配的位置当大于混淆阈值的实例数超出查找个数时，此时返回的实例未必按照得分高低进行返回

Note：当没有实例的分数大于混淆阈值时，其会按照接受阈值的高低进行返回，即此时混淆阈值是不起作用的。

# 7.LineMaxTool 基础应用

$\textcircled{1}$ 预期法线：在限定区域内搜索符合搜索方向的边缘点

# ②边缘点提取参数：

渐变内核大小（以像素为单位）：该工具使用该值来分析运行时图像并生成代表边缘信息的一组梯度向量。

当图像中感兴趣的功能显示锐利边缘时，在短距离内发生亮像素和暗像素之间的过渡时，请使用较小的值。增加此值可以使图像中的边缘更平滑。

投影长度（以像素为单位）：该值确定用于梯度场投影的区域数。较小的值可使工具以比较大的值更精细的粒度来分析图像，但是可能需要更多时间才能执行该工具。较大的值可以提高工具的执行速度，但可能无法检测到要工具定位的边缘。通常，使用的值至少应与“以像素为单位的渐变内核大小”设置的值一样大，但是如果工具继续定位所需的边缘点则可以将其设置为更高的值。

对比度阈值：指定边缘的最小对比度，该边缘被视为要定位的特征的候选边缘。增大此值可找到鲜明对比的边缘，并忽略亮像素和暗像素之间的过渡不那么突出的边缘，或者当要在图像中定位的边缘的对比度较小时，降低此值。

边缘的对比度是整个边缘的平均像素值的绝对差，范围为 0到255。该工具仅考虑对比度大于您指定的值的边缘。例如，将此值设置为10会拒绝对比度低于10个像素值的边缘点。

标准化对比度阈值：增大此参数的值可丢弃图像明亮区域中的虚假边缘点。请注意，将其设置为1会舍弃所有边缘点，从而使该工具实际上无法使用。

# 边缘约束：

极性：边缘的极性指向包含在行中。

角度公差：边缘点渐变方向与法线垂直的最大允许角度差。增大此值可使工具将更多的边缘点视为内点，从而更改找到的线段的位置。

距离公差：内线和拟合线之间的最大允许距离。随着您增加边缘距离公差，您可以允许该工具将更多边缘点视为内部并更改找到的线的位置。

# 线段约束：

最 大 行 数 ： 您 要 工 具 定 位 的 最 大 行 数 。 请 注 意 ， 仅 当 VisionPro 安 装 中 包 含VisionPro.LineMaxMulti 许可证时，编辑控件才支持此参数。有关 VisionPro 安全性的详细信息，请参阅VisionPro安全性密钥主题。将此值设置为0可以防止工具返回任何线结果，但仍会生成一组离群值供您的视觉应用程序使用。

角度公差：找到的线段的旋转量可能与您指定的预期角度不同。较低的值将强制工具定位与梯度搜索方向更平行的线段。

覆盖阈值：最小可接受线宽比与最大可能线束数或沿线段的边缘特征的最小比例。例如，将值指定为0.5，即使未在线的 $5 0 \%$ 上检测到边缘点，该工具也可以定位线段。同时使用 Coverage Threshold 和 Length Threshold 以满足您的线路配合要求。注意将值指定为 1.0会强制工具仅在没有间隙的地方找到线段时才定位线段。

长度阈值：设置找到的线段的最小可接受长度。

# 拟合算法：

穷举模式：对所有边缘点组合进行线拟合评估。

RANSAC模式：随机样本共识方法（RANSAC）：一种在存在许多候选边缘点的情况下对边缘点进行鲁棒拟合的算法。保证值会影响选择最佳边缘点进行线拟合的可能性。

# 9.CogBlobTool 高级参数

用于搜索斑点，即输入图像中任意的二维封闭形状，针对图像中二维封闭形状的检测与分析，根据用户设定好的灰阶范围来区分的，能够提供斑点的数量、位置、形状和拓扑等信息。

适用场景：二维图像的分析 高对比度(背景和对象对比明显) 有无检测 尺寸和方向会发生变化。

不适用场景：低对比度、在两种灰阶范围内不能突出主要特征 需要模板匹配。

$\textcircled{1}$ 分割和分析：分割是第一步，将 blob 区域找出来

图像分割：

$\textcircled{2}$ 模式：无 映射 硬阈值(固定) 硬阈值(相对) 硬阈值(动态) 软阈值(固定) 软阈值(相对)

减影图像 极性 阈值

硬阈值(固定)：用于分割最简单的方法是选择一个像素值作为阈值，低于阈值的灰度值的所有像素被作为目标像素，高于阈值的所有像素被指定为背景像素，这种技术被称为二进制阈值或硬阈值。会产生空间量化误差，产生空间量化误差的原因：分割出来的对象边界，不是刚好两个像素之间的边界。会影响 blob所提供的的结果中尺寸、周长和位置等。以 $50 \%$ 来分界空间量化误差会随着Blob的周长与Blob区域的比率而增加。

软阈值和像素加权：在硬阈值中将图像的像素分为对象和背景两种，优点是速度快，缺点是产生空间量化误差。为了消除空间量化误差，引入软阈值，软阈值将图像像素分为对象、边缘和背景三种。引入像素加权来表示可能性，像素加权使用 0~1表示像素，0是背景，1是对象， $0 <$ 像素加权 $^ { < 1 }$ 代表的是对象的边缘。软二进制阈值，将灰度图像分割成加权像素值的图像。软二进制阈值由阈值范围组成。值高于阈值范围的像素分配权重为0(背景)，值低于阈值范围的像素分配权重为 1(对象),在阈值范围内的值的像素权重为 0-1之间，通常以线性方式分配。

相对硬阈值：固定阈值的问题，无论是硬阈值还是软阈值，在光照亮度线性变化时都会出现无法分割的情况

设定：分两步： 设定低尾部和高尾部 设定阈值百分比(两个波峰之间的波谷的对应值就是我们需要设定的阈值)。使用相对硬阈值对亮度变化的图像实现完美分割。

动态硬阈值：无论是直接指定硬阈值还是成比例的计算阈值，都是指定一个阈值，以有效的从背景像素分割对象像素。是根据输入图像的直方图来计算合适的阈值。阈值是通过最小化在阈值两边的像素的权重方差来计算的。有双峰分步的直方图会出现很好的分割如果不是双峰分布，可能就不能得到很好的分割。

# 12. Cog3DRangeImageCrossSectionTool 应用

通过3D范围图像横截面工具：可以对从Cognex 3D位移传感器获取的范围图像的垂直切片进行测量。

两个步骤：该工具首先从距离图像中创建一个称为“轮廓”的垂直切片，然后该工具运行一系列的运算符来分析轮廓并进行所需的测量。

$\textcircled{1}$ 创建配置文件：要创建配置文件，您首先要在范围图像的 X-Y平面中选择一个矩形的“切片”区域。此矩形定义您的垂直切片，通常长而细。该工具会切掉该区域之外的所有范围数据，仅保留切片。切片的切割面沿切片矩形的整个边界平行于范围图像的 Z轴。然后，将切片的3D范围数据沿Z轴向下切割时，将其投影到该区域的中心线形成的平面上，从而将其展平为2维。该中心平面称为切割平面。扁平切片称为轮廓。它表示切割平面上平均范围数据的侧面视图。如果您在切割平面的点处对范围图像进行切片，然后从垂直于切割平面的视点查看数据，这将是您所看到的数字近似值。轮廓具有其自己的2D坐标空间集。这些空间应被视为存在于切割平面中，并且与范围图像的2D坐标空间完全不同。  
$\textcircled{2}$ 运行操作员：您可以创建一系列操作员，这些操作员将分析轮廓并进行测量。可以将每个操作员视为仅在配置文件上工作的特殊 VisionPro工具。每个运算符都有自己的参数，自己的结果以及针对这些结果的一组容差。操作员序列的行为就像一个小的工具组。

运算符的类型如下：

从轮廓中提取几何特征（点和线段）的运算符。

运算符从现有的几何特征中计算出新的几何特征。

在几何特征之间进行测量的运算符。

可以组合任意数量的这些运算符来实现所需的测量。
