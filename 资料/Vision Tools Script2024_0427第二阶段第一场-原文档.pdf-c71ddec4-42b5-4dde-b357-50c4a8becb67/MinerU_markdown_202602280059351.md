# Simple vs Advanced

# Simple 优势

使用起来简单

# Advanced 优势

更加灵活（可以动态添加终端，等等）

功能强大（可以对图像显示等进行更改）

可以动态禁用启用工具

等等

Simple Script 与 Advanced Script 区别：

1）Simple Script 没有记录覆盖这一模块，如果需要在脚本编辑器添加界面显示，必须是在 Advanced Script 编辑模式下才可以。

2）在脚本编辑器实例化工具类,代码编辑方式不相同；

3） Inputs/Outputs 输入输出，代码编辑方式也不相同；
