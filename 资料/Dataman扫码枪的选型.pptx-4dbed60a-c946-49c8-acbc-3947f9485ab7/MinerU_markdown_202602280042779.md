# 4. 扫码需求

了解了各型号扫码枪的特性后，就要了解扫码需求了，大致可以分以下几点：

用于自动化设备还是手动工站？

• 条码特性，如条码大小，规格，打码方式，打码材料等

安装方式，如有多大空间可供扫码枪安装调试？

• 公司主推产品，如以前是 DM302X, 现在是 DM262X

一次性扫一个码还是多个码等

综合以上因素，选择合适的扫码枪
