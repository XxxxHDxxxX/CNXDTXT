# 2.相机选型考量参数：检测精度 产品尺寸 是否飞拍（卷帘或者全局） 黑白彩色

视觉精度计算： 单方向视野范围/相机单方向分辨率

eg： 视野 $4 0 \mathrm { m m } ^ { * } 3 0 \mathrm { m m }$ 相机分辨率 $6 0 0 ^ { * } 4 8 0$

此时精度为 $4 0 \mathrm { m m } / 6 4 0 \mathrm { = } 0 . 0 6 2 5 \mathrm { m m }$

同样已知检测精度求分辨率时：

eg： 精度 $0 . 0 1 \mathsf { m m }$ 视野 $4 \mathsf { m m } ^ { \ast } 3 \mathsf { m m }$ 此时 $4 \mathrm { m m } / 0 . 0 1 \mathrm { m m } = 4 0 0$ 像素 分辨率为 $4 0 0 ^ { * } 3 0 0$ 就行

实际视觉测量时一般要求理论精度 $>$ 实际精度需求

eg：视野 $6 ^ { * } 4 \mathsf { m m }$ 物件为 $4 { \pm } 1 \mathsf { m m }$ 精度需求为 $0 . 0 2 \mathsf { m m }$

此时视野 $6 \mathsf { m m }$ 精度按照每四个像素对应 $0 . 0 2 \mathsf { m m }$ 则是 $0 . 0 0 5 \mathsf { m m }$ 的精度

分辨率： $6 \mathrm { m m } / 0 . 0 0 5 \mathrm { m m } = 1 2 0 0$ 像素 则选用1280x1024分辨率的相机即可

帧率:单位FPS，帧/秒 指相机每秒钟能采集多少幅图像 eg：15帧/s 表示相机一秒钟最大能采集15张图片

一般来说： 分辨率越大，帧率越低

最高帧率取决于采集速度，数据转换速度和数据传输速度

相同型号的相机在不同分辨率情况下采集最高帧率不相同，一般情况下CCD相机随着

像高减小而帧率增加，与像宽无关，

CMOS相机采集帧率与图像面积有关，面积越小，速度越快，但这两种情况都不会无限制

提高帧率，由于其他时序关系影响，达到一定高的帧率之后就不会改变了

噪声：相机芯片电路和信号传输电路中形成的干扰信号，导致图像中出现非实物反映的图像

曝光时间：电子快门打开采集信息的时间，曝光时间越长，图像越亮，同时抗震动能力越差，

对运动物体拍摄时曝光时间越长拖影越长，过长的曝光时间会使相机帧率下降

增益：将图像数字信息同时扩大，可以增加对比度和亮度，提高成像效果，噪声也随之扩大

像元深度：对于黑白相机 像元深度 定义灰度由暗到亮的灰阶数 工业上都是使用的8位的像元深度

eg：像元深度是 8 位的相机，输出的图像灰度等级是 2 的 8 次方，即 0~255 共 256 级，

像元深度是10位的相机，输出的图像灰度等级是2的10次方，即0~1023共1024级

像元深度越大，可以增强测量精度，但同时降低系统速度，所以应当谨慎选择

彩色和黑白相机区别： 黑白相机分辨率高 信噪比大 灵敏度高

彩色相机灵敏度相对较低，清晰度相对较低，所以除非需要检测彩色信息，否则采用黑白

# 操作方法：

1.构建相机像源，并采集一张较好的图像，然后再创建CogPMAlign工具，同时将其输入图像连接到像源  
2.配置 PMAlign 工具来定位图像中的物体，创建 PatInspect 工具，连接图像，将 PMAlign 工具的 GetPose输出端连接到PatInspect工具的Pose输入端，运行工具进行Pose信息的传递。

3.打开 PatInspect 工具编辑器，点击抓取训练图像和原点  
4.切换到当前训练图像，框选需要检测的区域  
5.点击训练新模式来创建一个训练模式  
6.继续运行工具以获取新图像，返回编辑控件注意新的当前输入图像，单击统计训练当前模式，将新图像合并到现有的训练模式中，此时运行参数选项卡中的训练模式图像窗口中的训练模式可能会发生轻微变化，此外，编辑控件将更新用于创建这个训练模式的图像数量。  
7.重复上述操作，对于添加的每个新图像，训练模式的变化也会越来越小，但最终应该得到一个训练模式，该模式将生成包含真实缺陷的阈值差分图像。  
8.当获得足够的图像来训练模式时，单击结束统计训练(End Statistical Training)，可结合其它工具进行进一步分析PS：创建好训练模式后，既可以用来检测连续运行的图像，但是注意不同图像可能会存在一些光照的变化，同时图像中的物体位置也可能轻微移动，因此需配置PatInspect工具获得能够涵盖这些细微变化的训练模式。

# 1.3CogCNLSearchTool：

作用：是定位或测量事先训练的一个或多个特征的质量，即用来测量当前图像中某一天特征与先前训练的图模型的相似度。

应用：

定位：通过定位物体上的特征来确定已知物体的位置和方向

存在/不存在检测：验证图像中存在期望数量的特征。

测量：测量长度。直径、角度和其他关键尺寸，通过定位图像中的特征，然后计算它们之间的距离。

缺陷检测：搜索图像中的缺陷。只有在预先知道缺陷的出现时才有用。

线性模式搜索：如果指定搜索图像的亮度与模式图像的亮度线性不同，则 CNLSearch将查找搜索图像中像素值的模式与模式图像中

像素值的模式最相似的部分。这种搜索称为强度相关搜索，因为搜索图像和模式图像之间的相似度是通过计算两幅图中

灰度像素值模式之间的相关系数来确定的。

CNLSearch用于计算两幅图像之间的相关系数的方法不受图像之间亮度的线性变化的影响。

非线性模式搜索：如果指定搜索图像的亮度与模式图像的亮度非线性不同，则 CNLSearch通过查找搜索图像中边缘和非边缘的图案

与模式图像中边缘和非边缘的图案最相似的位置来搜索图像中的团。

由于CNLSearch的非线性模式搜索的是边缘模式而不是像素值模式，因此 CNLSearch对模式图像和搜索图像之间的

线性和非线性亮度变化都免疫，只要亮度变化不影响搜索图像中的边缘模式。

工作原理：根据模式的训练图像在搜索图像中查找模式的位置。除了返回模式在搜索图像中的位置，它还通过返回分数来指示

搜索图像中的模式与训练图像中的模式匹配的程度。分数表示训练图像与返回其位置的图像之间的匹配程度。

得分范围从0.0，表示模式和特征之间没有相似性，到1.0，表示完全匹配。

除了返回在搜索图像中找到的最接近匹配的位置和分数，CNLSearch还可以返回图像中其他不太相似的模式实例的位置和分数。

即使使用高性能CPU，对于大多数实际应用程序，彻底搜索也太慢，它使用了一种更复杂的技术来定位图像中的特征。首先，

快速扫码图像以定位可能匹配的位置，然后只对这些候选位置进行相似性评估，返回最佳匹配的位置。
