# 4.6读取多个码设置

点击-设置-读码类型图标-多代码

# 4.6.1.多代码设置

代码数量_-需要读取几个码，如果两个就写2，兰个就写3

允许部分结果输出

如果视野内物料码没有全部解出来，要不要只把解出来的输出，除非用脚本或者OEM程序能通过信隱省岂判断一般不打勾

# 4.6.2.物料码制个数

需要读取的物料码有几种码制，每种码制各多少

如果代码数量为2，且都是DataMatrix，则需要将DataMatrix的代码最天数量改为2

，下面其它码值数量改为2

如巢代码数量为3

且DataMatrix的数量为2，QR为1，则需要将Data Matrix 的代码最大数量改为2，QR的代码最大数量为1，其它码值数量改

# 4.6.3.输出顺序

.从左到右

按照读取码的位置按照从左到右排序，当选择此位置然后点击反向可以改为从右到左.从上倒下

按照读取码的位置按照从上到下排序，当选择此位置然后点击反向可以改为从下到上

注意：

输出顺序要选择然后点击向上置顶

# 七．常见异常与处理方法

1．解码失败 $\twoheadrightarrow$ 调整配置

2．扫码枪连接不上 $\twoheadrightarrow$ 查看网线是否连接好，IP是否正确

3．扫码枪打不开卡死 $\twoheadrightarrow$ 断电重启

4.当读取设置有多个通道，解码时无法切换设置通道时须手动恢复出厂设置，手动配置参数，不要导入当前设备备份的配置文件
