# Training Contents:

a) Geometry – Creation 创建   
b) Geometry – Intersection 相交  
c) Geometry - Measurement 测量

# CogTool-Geometry - Creation

> CogCreateCircleTool // 创建圆  
> CogCreateEllipseTool // 创建椭圆  
>CogCreateGraphicLabelTool // 创建显示  
> CogCreateLineBisectPointsTool // 创建两点的平分线  
> CogCreateLineParallelTool // 以某一点创建某线的平行线  
> CogCreateLinePerpendicularTool // 以某一点创建某线的垂线  
> CogCreateLineTool // 创建线（以某点和角度创建线）  
> CogCreateSegmentAvgSegsTool // 创建两条线段的平均线  
> CogCreateSegmentTool // 创建线段

# CogTool-Geometry - Intersection

CogIntersectCircleCircleTool // 检测圆与圆相交   
CogIntersectLineCircleTool // 检测线与圆相交   
CogIntersectLineEllipseTool // 检测线与椭圆相交   
CogIntersectLineLineTool // 检测线与线相交   
CogIntersectSegmentCircleTool // 检测线段与圆相交   
> CogIntersectSegmentEllipseTool // 检测线段与椭圆相交   
CogIntersectSegmentLineTool // 检测线段与线相交   
CogIntersectSegmentSegmentTool // 检测线段与线段相交

# CogTool-Geometry - Measurement

CogAngleLineLineTool // 测量线与线的角度  
CogAnglePointPointTool // 测量点与点的角度  
CogDistanceCircleCircleTool // 测量圆与圆的距离   
CogDistanceLineCircleTool // 测量线与圆的距离   
CogDistanceLineEllipseTool // 测量线与椭圆的距离   
CogDistancePointCircleTool // 测量点到圆的距离   
CogDistancePointEllipseTool // 测量点到椭圆的距离   
CogDistancePointLineTool // 测量点到线的距离   
CogDistancePointToPointTool // 测量点到点的距离   
CogDistancePointSegmentTool // 测量点到线段的距离   
> CogDistanceSegmentCircleTool // 测量线段与椭圆的距离   
> CogDistanceSegmentEllipseTool // 测量线段与椭圆的距离   
CogDistanceLineTool // 测量线段与线的距离   
> CogDistanceSegmentSegmentTool // 测量线段与线段的距离
